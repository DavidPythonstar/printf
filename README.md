The _printf() function

We are creating a variadic function that performs exactly the same purpose just like the printf() in C programming.

The printf() is  a function in C programming used to do formatted printing.
It comprises of the following components: 

ARGUMENTS 
	Strings
	Floats
	Boolean
	Integers
	Characters
	Doubles
FORMAT SPECIFIERS
	Special characters
	Precisions


Unlike other functions with a fixed number of arguments, the **printf()** function takes on an infinite number of arguments. Functions like this in C language are known as **Variadic functions**.
C programming permits programmers to create such functions with the help of the
Standard header file **stdarg.h**. 
This library provides a mechanism of accessing arguments passed to the function that accepts a variable number of arguments.
